# Assumption-Evidence Table

| Team Assumption | Evidence from Respondents |
|---|---|
| Attendance takes too much class time | Multiple students and lecturers reported 10–20 minutes spent on attendance. |
| Attendance records contain errors | Several students reported missing or incorrect attendance records. |
| Students want a faster method | Most respondents were willing to try a QR attendance system. |
| Manual attendance creates queues | Students reported delays before class starts. |
