# Case Study Brief: Smart Classroom Attendance System

## Context

Many universities still use manual attendance methods such as paper sign-in sheets, verbal attendance checks, or spreadsheet records. These methods are time-consuming, prone to errors, and can disrupt classroom activities. Students may forget to sign attendance sheets, while lecturers spend valuable teaching time managing attendance records.

## Target User Groups

- Students attending classes
- Lecturers and instructors
- Academic administrators responsible for attendance records

## Initial Problem Assumption

Students and lecturers experience difficulties with manual attendance processes because they are inefficient, prone to errors, and require unnecessary administrative effort.

## Current Alternatives

- Paper attendance sheets
- Verbal attendance checking
- Spreadsheet-based attendance tracking
- Manual attendance reports

## Possible MVP Direction

A web-based Smart Classroom Attendance System that uses QR codes to record attendance. Students scan a QR code when entering class, while lecturers can monitor attendance records through a dashboard and generate reports automatically.

## Feasible Tools

- HTML, CSS, JavaScript
- QR Code Generator API
- Google Forms
- Google Sheets
- Firebase
- Figma
- Simple Web Dashboard
