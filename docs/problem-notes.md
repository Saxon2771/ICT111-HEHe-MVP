# Problem Notes

## Target Users
- University students
- Lecturers
- Academic staff

## Main Problem
Attendance is currently recorded manually using paper sheets or verbal confirmation.

## Pain Points
- Students forget to sign attendance.
- Attendance sheets are lost or damaged.
- Long queues before class.
- Lecturers spend class time checking attendance.
- Attendance records contain errors.

## Current Alternatives
- Paper attendance sheets
- Verbal attendance
- Spreadsheet tracking

## Consequences
- Students marked absent incorrectly.
- Time wasted during class.
- Administrative workload increases.
