# Selected Opportunity

## Project Title
Smart Classroom Attendance System

## Target User
- University students
- Lecturers
- Academic administrators

## Problem Statement
Many universities still use manual attendance methods such as paper sign-in sheets or verbal attendance checks. These methods are time-consuming, prone to errors, and make attendance records difficult to manage and monitor.

## Current Alternative
- Paper attendance sheets
- Manual attendance checking by lecturers
- Spreadsheet-based attendance records

## Proposed IT Solution
Develop a Smart Classroom Attendance System that allows students to check in using a QR code generated for each class session. Lecturers can automatically record attendance, monitor student participation, and generate attendance reports through a web-based dashboard.

## MVP Type

### Web Application Prototype

**Core Features**
1. Student QR Code Check-in
2. Attendance Record Management
3. Lecturer Dashboard
4. Attendance Report Generation
5. Basic User Authentication

## Why This Opportunity Was Selected

The Smart Classroom Attendance System received the highest score in the NUF (New, Useful, Feasible) evaluation. It addresses a common problem experienced by both students and lecturers, provides clear value to users, and can be developed as a feasible MVP within the semester timeline using web technologies.
